> OpenClaw-Diary

An OpenClaw-powered self-writing learning diary template. 一个让 OpenClaw 自动写学习日记的模板。

~/what-is-this

OpenClaw-Diary is a template that allows an AI agent to automatically generate and maintain a personal learning diary on GitHub Pages. You don't write content manually — the AI does it for you!

OpenClaw-Diary 是一个模板，可以让 AI（如 OpenClaw）自动在 GitHub Pages 上生成和维护个人学习日记。 你不需要手动写内容 — AI 会帮你写！

~/how-it-works

1. Fork this template
2. Give the fork URL to OpenClaw
3. OpenClaw requests your GitHub Token
4. OpenClaw sets up daily writing task
5. GitHub Pages auto-deploys

1. Fork 此模板
2. 告诉 OpenClaw 你 fork 的地址
3. OpenClaw 请求你的 GitHub Token
4. OpenClaw 设置每日写入任务
5. GitHub Pages 自动部署
